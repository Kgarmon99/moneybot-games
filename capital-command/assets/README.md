# Capital Command Assets

This game uses lightweight canvas-drawn MoneyBot primitives instead of generated bitmap art for fast loading and crisp scaling.

- MoneyBot coach/avatar: CSS and canvas shapes based on the canonical MoneyBot mascot reference at `references/mascot/canonical-operator-reference.png`.
- Cashflow coin: canvas-drawn green/gold income token.
- Expense bill: canvas-drawn red obligation token.
- Debt chain: canvas-drawn linked pressure token.
- Emergency shield: canvas-drawn blue-green shield effect.
- Compound engine: canvas-drawn investment node with rotating core.
- Risk storm: canvas-drawn volatility token.

No third-party or hotlinked assets are used.
