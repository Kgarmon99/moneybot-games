## Design System Decisions
- **Robinhood Theme:** White background (`#FFFFFF`), neon green accents (`#00C853` or `#00E676`), slate-grey text. This is the preferred premium aesthetic.
- **Mobile-First:** All games must work on iOS Safari. Key fixes applied: deferred AudioContext initialization, DOM queries wrapped in DOMContentLoaded, global function exposure for onclick handlers.
- **Cache Busting:** Use `?v=` query params when updating files for mobile browsers.
